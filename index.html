A Better Language

Welcome to the project page for the compiler for the Boh programming language! Boh is a language in many ways designed to be ideal for just about any programmer with a classic object-oriented mind, and manages to do stuff right where a lot of other languages fail.

What's so special about Boh?

Boh is a language heavily influenced by C# and Java, and any C#/Java programmer will probably feel right at home writing Boh code.

But hold on, Boh isn't just-another-Java-cloneâ„¢, it actually does some things well where C#/Java have failed. Boh is special in the following:

  • Boh has a number of generators, including, but not limited to:
    • A native code generator, via LLVM or C
    • A JavaScript code generator, for native web applications
  • Boh is designed to both have a comfortable command line interface, and provide easy IDE-integration
  • Boh is 100% free and open-source, no price, everything is open, anyone can contribute
  • Boh is more complete than C#/Java were when they were released, allowing for a cleaner standard library
  • Boh is cross-platform natively, and at the moment supports (at least) Windows and Linux, and many more through JavaScript

If that doesn't make Boh different from C#/Java in the slightest, then I don't know what will.
